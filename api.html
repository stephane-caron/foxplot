API#

Plot time-series data from line-delimited JSON.

class foxplot.Fox(from_file=None, time=None)#

Frequent Observation diXionaries, our main class.

Our main class to read, access and manipulate series of dictionary data.

detect_time()#

Search for a time key in root keys.

Return type:

None

get_series(label)#

Get time-series data from a given label.

Parameters:

label (str) – Label to the data in input dictionaries, for example /observation/cpu_temperature.

Return type:

Series

Returns:

Corresponding time series.

property labels: List[str]#

List of all labels present in the data.

plot(left, right=None, title=None, left_axis_unit='', right_axis_unit='', print_command_line=False)#

Plot a set of indexed series.

Parameters:

  • left (Union[Series, Node, List[Union[Series, Node]]]) – Series to plot on the left axis.

  • right (Union[Series, Node, List[Union[Series, Node]], None]) – Series to plot on the right axis.

  • title (Optional[str]) – Plot title.

  • left_axis_unit (str) – Unit label for the left axis.

  • right_axis_unit (str) – Unit label for the right axis.

  • print_command_line (bool) – If true, print out how to obtain the plot from the command line.

Return type:

None
